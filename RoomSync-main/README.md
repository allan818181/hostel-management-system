# RoomSync 🏨

RoomSync is a modern Hostel/Room Management System designed to simplify accommodation administration. It helps manage rooms, tenants, and bookings efficiently, providing an organized platform for hostel or dormitory operations.

## Features ✨
- Add, update, and manage rooms
- Manage tenants and their details
- Track bookings and room availability
- User-friendly interface for admins
- Generate reports for hostel operations

## Tech Stack 💻
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP 
- **Database:** MySQL
